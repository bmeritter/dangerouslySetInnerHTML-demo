# dangerouslySetInnerHTML-demo

1. `git clone git@github.com:bmeritter/dangerouslySetInnerHTML-demo.git`

2. `cd dangerouslySetInnerHTML-demo`

3. `npm run webpack`

4. `npm start`

5. 访问：`localhost:3000`
